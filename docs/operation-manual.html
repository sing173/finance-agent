1. 准备工作（测试阶段已内置，可跳过）

第一次使用前，需要完成两项一次性配置。后续日常操作无需重复。

导入科目表

  1. 从金蝶精斗云导出科目表（xlsx 格式）
  2. 点击主界面「导入科目」按钮，选择导出的文件
  3. 系统内置约 300 条标准科目，导入后将与内置科目合并

配置银行账号映射

让系统自动识别每笔交易的银行科目：

操作说明
新增选择银行代码 → 匹配方式（后缀/精确）→ 输入账号模式 → 选科目
精确匹配账号完全一致才匹配（适用于对公账户）
后缀匹配账号末尾匹配指定模式即可（如尾号 4363 → 工行基本户）
修改/删除在「账号管理」表格中直接编辑或删除
提示：账号映射配置完成后，解析流水时系统会自动将银行账号替换为对应的会计科目（如"银行存款_工行基本户"），无需手动选择。

Step 1：解析银行流水

支持拖拽、浏览选择或批量导入文件。

  • 自动识别：系统检测银行和文档类型。识别失败时弹出手动选择窗口，指定银行 + 文档类型后重试

  • OCR 扫描件：如果PDF是扫描件/图片，则自动触发 OCR（约 10–20 秒/页），耗时较长

Step 3：检查与修正科目

解析完成后，进入凭证预览面板检查科目分配结果。

修正未匹配科目的操作：

  1. 点击红色高亮条目 → 弹出科目选择器
  2. 搜索关键词或按类别筛选 → 点击选择
  3. 选择后自动更新，匹配来源变为 manual

如果大量条目科目相同，可使用批量填充：上方下拉菜单选择常用预设科目，一键应用到所有未匹配条目。

分录移动与拆分

凭证预览支持对分录进行重组和拆分，适应复杂记账场景：

功能操作方式适用场景
移动分录勾选一条或多条非银行分录 → 底部选择目标凭证 → 自动移动并重新编号将误归类的分录调整到正确凭证
拆分分录勾选单条分录 → 点击「拆分」→ 输入拆分金额和对应科目 → 生成两条新分录一笔交易涉及多个费用科目时拆分记账

Step 4：确认导出

  1. 检查所有条目均有科目（无红色 ⚠ 标记）
  2. 点击「确认导出」→ 选择保存位置
  3. 导出文件为金蝶精斗云2026xxxxxx，直接导入金蝶精斗云即可

3. 常见问题

Q1: "无法识别银行类型"怎么办？
  • PDF 文件可能不是当前支持银行的格式
  • 扫描件需 OCR 识别（约 1–2 秒/页），请耐心等待
  • 手动选择银行 + 文档类型即可强制解析
Q2: 科目匹配不准确怎么办？
  • 在凭证预览中手动修正 → 系统自动学习
  • 修正次数越多，历史匹配越准确
  • 可在「账号管理」中完善银行账号映射规则
Q3: 支持哪些文件格式？
格式说明
PDF银行流水 / 回单（文本型 + 扫描件 OCR）
CSV工商银行对账流水（GBK 编码）
Excel招商银行流水
