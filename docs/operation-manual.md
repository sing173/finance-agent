# FinanceAssistant 操作手册

---

## 一、快速开始

**一句话流程：** 拖文件 → 自动识别 → 解析 → 确认凭证 → 导出

1. 启动应用，看到拖拽区域
2. 拖入银行的流水 PDF/CSV/Excel 文件
3. 点击「开始解析」
4. 点击「凭证生成」→ 预览确认科目
5. 点击「确认导出」→ 存为金蝶可导入的 Excel

---

## 二、单文件处理

### 2.1 选择文件
- **拖拽：** 把文件拖到中间拖拽区域
- **浏览：** 点击拖拽区域，从文件夹选择
- ⚠ 支持格式：PDF（流水/回单）、CSV（工行）、Excel（招行）

### 2.2 银行检测
- 系统自动识别银行和文档类型
- 识别成功 → 显示「工商银行·流水」之类提示
- 识别失败 → 弹出手动选择窗口，选银行+文档类型后重试

### 2.3 查看交易明细
- 解析完成后自动展示交易列表
- 支持排序、筛选、分页
- 顶部显示 收入/支出 统计摘要

### 2.4 常见问题
| 问题 | 解决办法 |
|------|---------|
| 提示"无法识别银行类型" | 手动选择对应的银行和文档类型再试 |
| 解析结果为空 | 确认文件格式是否在支持列表中 |
| 金额/日期解析错误 | 确认 PDF 是否为扫描件（OCR 需额外时间） |

## 三、批量文件处理

### 3.1 添加多个文件
- 一次拖入或多个文件同时选择（默认最多 5 个）
- 文件列表自动去重
- 可随时移除不需要的文件

### 3.2 批量检测
- 点击「识别文件」→ 系统逐个识别银行类型
- 每个文件旁显示检测状态（蓝=已检测，紫=已设置）
- 检测失败的文件可点击「修改配置」手动指定

### 3.3 批量解析
- 点击「开始解析」→ 逐文件解析
- 解析完成后显示摘要：共 X 个文件、成功/失败数量、总交易笔数
- 每个文件可展开查看交易明细，失败文件显示错误信息并可重试

### 3.4 失败处理
- 失败的文件显示红色，点击可修改配置后重试
- 成功文件可合并导出统一凭证

## 四、账号与科目管理

### 4.1 银行账号→科目映射
- 入口：主界面「账号管理」按钮
- **新增：** 选择银行代码 → 匹配方式（后缀/精确）→ 输入账号模式 → 选科目
- **修改/删除：** 表格中直接操作
- 作用：解析时自动识别银行账号对应的会计科目

### 4.2 导入科目表
- 首次使用需导入金蝶导出的科目表（xlsx 格式）
- 入口：主界面「导入科目」按钮
- 导入后系统内置约 300 条标准科目

## 五、凭证生成与导出（核心功能）

### 5.1 进入凭证预览
- 解析完成后点击「凭证生成」
- 进入全屏预览面板

### 5.2 凭证预览面板说明
每条凭证包含：

```
凭证 #1  日期: 2026-03-31
  分录1  借  管理费用_物业管理费    ¥1,200.00
  分录2  贷  银行存款_工行基本户    ¥2,400.00
         ↑ 匹配来源: rule（规则匹配）
```

- 未匹配科目 → **红色高亮** ⚠
- 同类别交易自动合并为一张凭证

### 5.3 手动选择科目
- 点击红色高亮条目 → 弹出科目选择器
- 搜索关键词或按类别筛选 → 点击选择
- 选择后自动更新

### 5.4 批量填充
- 上方下拉菜单提供常用预设科目
- 选择后一键应用到所有未匹配条目

### 5.5 保存/加载草稿
- **保存草稿：** 未完成可暂存，下次继续
- **加载草稿：** 从历史草稿列表选择恢复
- 草稿跨会话不丢失（存在本地数据库）

### 5.6 确认导出
- 检查所有条目均有科目
- 点击「确认导出」→ 选择保存位置
- 导出的 Excel 可直接导入金蝶精斗云

## 七、常见问题（FAQ）

### Q1: 支持哪些银行和文件？
| 银行 | 流水 | 回单 | CSV | Excel |
|------|------|------|-----|-------|
| 工商银行 | ✅ | ✅ | ✅ | — |
| 招商银行 | ✅ | ✅ | — | ✅ |
| 广发银行 | ✅ | — | — | — |

### Q2: "无法识别银行类型"怎么办？
- PDF 文件可能不是支持银行的格式
- 扫描件需 OCR 识别（约 1-2 秒/页）
- 手动选择银行+文档类型即可强制解析

### Q3: 科目匹配不准确怎么办？
- 在凭证预览中手动修正 → 系统自动学习
- 用得越多，自动匹配越准（仅记录手动修正，不记录自动结果）

### Q4: 导出的 Excel 怎么导入金蝶？
- 金蝶精斗云 → 凭证管理 → 导入 → 选择导出的文件
- 自动匹配日期、凭证字、科目、金额

### Q5: 数据存在哪里？
- 所有数据存储在本地电脑，不上传云端
- 数据库位置：`%APPDATA%/FinanceAssistant/data.db`
