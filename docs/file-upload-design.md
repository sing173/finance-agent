# 文件上传方案决策记录

> 来源：grill-with-docs 会话，2026-05-15
> 状态：已对齐，待实现

---

## 核心流程

### 单文件模式

```
选文件 → 后端自动检测银行+格式 → 结果卡片展示"检测到：招商银行 · 账务明细清单"
         → 用户确认解析 → 解析成功 / 失败
```

### 批量模式（B 方案 + 跳过继续）

```
[批量解析] 入口 → 多选文件（最多5个） → 前端过滤格式 → 后端并行检测银行
→ 文件列表（文件名 + 检测状态）
→ fallback 面板（银行 + 表格类型 + OCR）
→ 逐个解析，跳过失败继续
→ 按文件分组折叠展示 + 全部导出
```

| # | 问题 | 决策 |

| 1 | 检测失败/猜错银行时是否有覆盖路径 | **有**——fallback 面板让用户手动选择 |
